# 달러보험 가입 제안서

인플레이션 시대, 자산을 지키는 가장 확실한 방법 - 달러보험

## GitHub Pages 배포

이 프로젝트는 GitHub Pages로 배포할 수 있습니다.

### 배포 방법

1. GitHub 저장소에 파일 업로드
   - `index.html` 파일을 저장소 루트에 업로드

2. GitHub Pages 활성화
   - 저장소 Settings > Pages
   - Source: Deploy from a branch
   - Branch: main (또는 master)
   - Save

3. 배포 확인
   - 몇 분 후 `https://synitart.github.io/Dollar/` 에서 확인 가능

## 주요 기능

- 인플레이션 대응 전략 설명
